
# 💼 Job Portal (MERN Stack)

A full-stack Job Portal web application built using the MERN stack that connects job seekers, recruiters, and admins. Users can search and apply for jobs, recruiters can post and manage jobs, and admins can control the overall system.

## 🚀 Features

### 🔐 Authentication & Authorization
- User Registration & Login  
- JWT-based Authentication  
- Password Encryption using bcrypt  
- Protected Routes  
- Forgot Password (Email verification using Mailtrap)  


## 👥 Role-Based Access
- Job Seeker  
- Recruiter  
- Admin  


## 🧑‍💻 Job Seeker Functionalities
- Create and Update Profile  
- Search Jobs by keyword  
- View all available jobs  
- Apply for jobs  
- Track applied jobs  

## 🏢 Recruiter Functionalities
- Post new jobs  
- View posted jobs  
- Update job details  
- Delete jobs  
- View applicants  

## 🛡️ Admin Functionalities
- Manage all users (view / delete)  
- Manage all jobs  
- Monitor applications  
- Control platform activities  
- Approve / Reject job postings *(if implemented)*  

## 🔎 Job Management
- Dynamic job listings  
- Job details page  
- Search functionality  

## 🛠️ Tech Stack

### Frontend
- React.js  
- HTML5  
- CSS3  
- JavaScript  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### Other Tools
- JWT (Authentication)  
- bcrypt (Password hashing)
